# Tags

Tags are a flexible way to organize snippets across folders. Unlike folders, a snippet can have multiple tags at the same time.

## Adding a Tag

Select the snippet, then enter a tag name in the **"Add Tag"** field below the snippet name.

::: info
If a tag already exists, it will appear in a dropdown for you to select. Tags are unique and cannot be repeated.
:::

## Deleting Tag

Right-click the tag and choose **"Delete"**.

::: warning
When you delete a tag, it will also be removed from all snippets that use it.
:::
