# Folders

Folders help you keep notes organized by project, topic, or area of work. Notes supports hierarchical folders, so you can build a simple tree instead of keeping everything in one long list.

## Adding a Folder

- Click the **"+"** button next to **"FOLDERS"** in the sidebar.
- Select **"File"** > **"New Folder"** from the menu bar.
- Press <kbd>Shift+Cmd+N</kbd> on macOS or <kbd>Shift+Ctrl+N</kbd> on Windows or Linux.

## Moving to Folder

Drag one folder onto another to create a nested structure.

## Renaming and Deleting

Right-click the folder and choose **"Rename"** or **"Delete"**.

## Folder Icons

Folder icons make larger note libraries easier to scan. Right-click a folder and choose **Set Icon**.

Use the **Icons** tab to search the built-in Material and Lucide collections. Choose **Remove Icon** from the folder context menu to restore the default icon.

### Emoji and Uploaded Images

<AppVersion text=">=5.9" />

- **Emoji** uses the system emoji style for your platform.
- **Upload** accepts JPG or PNG images up to 10 MB and shows a preview before applying the image.

Uploaded images are cropped from the center, resized to 128×128, and stored as a hidden `.icon.png` file inside the folder.

## Moving Notes

Drag and drop notes between folders.
