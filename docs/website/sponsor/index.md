## What massCode is

massCode helps developers organize, find, and reuse technical knowledge: snippets, notes, saved requests, calculations, and everyday utilities.

Unlike cloud-based alternatives, massCode stores user content locally as plain Markdown files. Developers keep ownership of their data, can sync it with the tools they already trust, and avoid being tied to a proprietary backend.

## Why it matters

- **Local-first by design.** No accounts, no telemetry, no cloud dependency. User data stays on the user's machine.
- **Privacy-respecting.** massCode aligns with developers who want useful software without behavioral tracking.
- **Open source and AGPL-licensed.** The app is auditable, forkable, and maintained in public.
- **A daily-use developer tool.** massCode is not an invisible dependency. It is a desktop app developers open directly during work.
